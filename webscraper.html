WebScraper

This project was done in my spare time, as I was interested in coding a web scraper for some time. I've looked at some websites but ended up choosing eBay as my scraping target.

Idea

The idea was simple. Build a program to grab product information from eBay and put it in an Excel file.
There are multiple ways to do this, but I decided on using Python and a popular package named 'Beautiful Soup'. This package makes it far easier to parse the HTML and XML documents and extract the data.

Coding

The project was done using VS Code wich I am fairly familiar with. However, there were a few concepts new to me, such as parsing through HTML, finding the correct information as well as storing it in an organised manner.
I used the inspect tool every browser has, and find my data on the HTML files. The code can also extract data from multiple pages, meaning thousands of products as each page displays around 60.
I also had to 'sculpt' the information as there were things, say, in the title that I did not need and restricting the length of certain data categories. The filtering of useless information was barely surface level.

Conclusion

This project reminded me of the Python syntax as I haven't worked with the language in some time. But, it was also a great opportunity to learn new concepts, which I plan to use on a similar web scraper, with a different target website, and hopefully better information filtering.

Files

The code is available on a GitHub repository. There is no documentation to go with it, as the project is not the same size as some of the others. It does, however, have comments throughout the code and should indicate what each part of the code is for.
